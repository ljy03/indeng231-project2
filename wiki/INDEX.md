# Golf Knowledge Base

## Overview

This wiki is a curated personal knowledge base covering all major aspects of golf — from the fundamentals of how the game is played, to the mechanics of the swing, to the sport's four major championships and its greatest players. Articles are interlinked with [[wiki-links]] so you can navigate from any concept to related topics. The knowledge base contains 15 articles totalling over 10,800 words, compiled from authoritative sources and organized for easy reference and Q&A.

## How to Use

Navigate by clicking any [[wiki-link]] to jump to that article. Use the Q&A system (`python run.py --qa`) to ask free-form questions — the system finds the most relevant articles and answers using their content. To add new topics, drop a raw source file into `raw/` and re-run `python run.py --compile`.
